# Проект автоматизации тестирования сервиса Самокат
Основа для написания автотестов — фреймворк pytest \
Команда для запуска — pytest -v \
Команда для просмотра отчёта о тестировании — allure open allure-report 

**Описание тестов** 

Навигация по кнопкам в шапке сайта - **test_navigation_buttons.py** 

**test_scooter_logo_redirect_to_main_page** # Проверка перехода на главную странцу при клике по логотипу Самокат  

**test_yandex_logo_opens_dzen_in_new_tab** # Проверка открытия нового окна с сайтом Дзен при нажатии на логотип Яндекса 

Проверка ответов раздела "Вопросы о важном" на главной странице - **test_question_list.py** 

**test_question_answer_opens** # Проверка корректности отображения ответов на "Вопросы о важном" по клику на вопрос  

Проверка успешного заказа самоката - **test_scooter_order.py** 

**test_scooter_order_pozitive** # Проверка успешного заказа самоката с заполнением всех обязательных полей двумя наборами данных
